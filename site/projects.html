Stray Aid Map

台灣每年有超過10萬隻流浪動物等待救援。Stray Aid Map 是全台第一個由青年主導的開放資料救援平台，整合收容所、動物醫院與餵食站資訊，讓每一隻動物都被看見、每一位愛心人士都找得到資源。

Taiwan has 100,000+ stray animals awaiting rescue each year. Stray Aid Map is Taiwan's first youth-led open data rescue platform, integrating shelters, veterinary clinics, and feeding stations so every animal is seen and every caring person finds help.
